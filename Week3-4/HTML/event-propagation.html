Understanding Event Handling

Events are things that can happen to HTML elements. When JS is used in HTML pages, it can react to these events

When nested elements each have an event listener, they cannot occur simultaneously, so we must signify the order in which they occur -- known as event propagation.
Bubbling - target element --> root/parent element
Capturing - root/parent element --> target
In the document.getElementById.addEventListener(event, function(), useCapture) , the "useCapture" parameter takes in a boolean referring to this propagation value. True = capture; False = bubble
